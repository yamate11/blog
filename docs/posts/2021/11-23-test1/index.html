$H\times W$ のグリッドの各マス$(i,j)$に， 1から9までの数字 $c_{ij}$ が書かれている． 以下を繰り返して文字列を作る．

  1. 行 $i$ を選ぶ．
  2. 列 $j$ を選び，$c_{ij}$ を文字列に追加する．
  3. 行 $i$ を選び直し，$c_{ij}$ を文字列に追加する．
  4. 1に戻る．

1,2 を $N$ 回繰り返してできる文字列の種類を mod 998244353 で求めよ．

制約: $2 \leq H,W \leq 10$, $1 \leq N \leq 300$

時間制限: 3秒

$S \subseteq \{1, \ldots, H\}$， $T \subseteq \{1, \ldots, W\}$ として，

$\textrm{dp}[i][0][S] :=$ 長さ $2i$ の文字列であって，この方法で 作った時に，最後にいる可能性のある行全体の集合が $S$ となるものの数

$\textrm{dp}[i][1][T] :=$ 長さ $2i + 1$ の文字列であって，この方法で 作った時に，最後にいる可能性のある列全体の集合が $T$ となるものの数
